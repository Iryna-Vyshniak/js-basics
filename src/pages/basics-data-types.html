2. Результати логічних виразів

2 && 0 && 3

Повертає: 0.

Оператор І (&&) зупиняється на першому хибному значенні (falsy). 0 - це falsy.

2 || 0 || 3

Повертає: 2.

Оператор АБО (||) зупиняється на першому істинному значенні (truthy). 2 - це truthy.

2 && 0 || 3

Повертає: 3.

Спочатку обчислюється `2 && 0` (результат 0). Потім `0 || 3` (результат 3).
